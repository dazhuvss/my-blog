## 回溯

### 模板

```java
class Solution {
    List<List<Integer>> ans = new ArrayList<>();
    List<Integer> path = new ArrayList<>();

    public List<List<Integer>> solve(int[] nums) {
        boolean[] used = new boolean[nums.length]; // 或 start 参数
        backtrack(nums, used);
        return ans;
    }

    void backtrack(int[] nums, boolean[] used) {
        if (path.size() == nums.length) {
            ans.add(new ArrayList<>(path));
            return;
        }
        for (int i = 0; i < nums.length; i++) {
            if (used[i]) continue; // 剪枝
            path.add(nums[i]);     // 选择
            used[i] = true;
            backtrack(nums, used); // 递归
            path.remove(path.size() - 1); // 回溯
            used[i] = false;
        }
    }
}
```

## 刷题节奏建议

1. 每天一个专题，先做 3 道基础 + 2 道综合。
2. 第二天先复盘昨天错题，再开新专题。
3. 每周做一次「盲写模板」：回溯、二分、滑窗、二叉树 DFS、DP。

把思路写成自己的话，比收藏题解更有价值。坚持复盘，进步会很明显。
